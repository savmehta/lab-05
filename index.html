JavaScript

Term 1
It's a programming language
Term 2
It's a ’virtual machine’ - a program ('run-time environment’)
Term 3
You can install on any computer (including mobile phones) that can run any programs written in Java
Term 4
Those programs will be safely ring-fenced (‘sandboxed’) so that nothing they do can affect the working of the rest of your computer
Term 5
And thirdly, it's the set of programs for developers to use that makes all that possible. Java is capable of doing far more complex applications than JavaScript, eg Java is used for the games on mobile phones and web-based applications like Google Docs
